FRCContour Model Summary


rev9.0 — March 2026 · M sandbox



🌸 FRC Model — คลอรีนอิสระคงเหลือ


FRC สลายตัวตามเวลาที่น้ำเดินทางในท่อ ใช้สมการ First-Order Decay:


FRC(t) = FRC₀ × e(-K × t)


ตัวแปรความหมาย


FRC₀ค่า FRC ที่สถานีต้นทาง


Kอัตราการสลายตัว (1/ชม.) — แต่ละสถานีต่างกัน, fit จากข้อมูลจริง 3 เดือน


tเวลาเดินทางของน้ำ (Travel Time, ชม.)


Blend Formula: Forecast = fOld×(1-frac) + fNew×frac
fOld = decay จากค่าจริงปัจจุบัน, fNew = decay จาก source ย้อนหลัง



⚡ EC Model — ค่าการนำไฟฟ้า


EC ไม่สลายตัวระหว่างเดินทาง → ค่า EC ปลายทาง ≈ EC ต้นทาง (delay ตาม travel time)



📊 ค่า K (Decay Rate)



ลำดับการใช้: Auto-fit K (จาก user) → STATION_K_OVERRIDE (fit 3 เดือน 61 สถานี) → Default by zone → 0.008

Auto-fit: Grid search K=0.001–0.200 → เลือก MAE ต่ำสุด → บันทึกใน localStorage → มีผลทั้ง popup chart และ report




🗺️ เส้นทางน้ำ (Source Mapping)



กลุ่มเจ้าพระยา (CP):

  • โรงผลิตน้ำบางเขน → TR1 → สจ.ลุมพินี (6ชม.), สจ.พหลโยธิน (5ชม.), สจ.สำโรง (7ชม.) → monitor

  • โรงผลิตน้ำบางเขน → TR2 → สจ.ลาดพร้าว (3ชม.), สจ.คลองเตย (4ชม.) → monitor

  • โรงผลิตน้ำบางเขน → TR3 → สจ.บางพลี (13ชม.), สจ.มีนบุรี (7ชม.), สจ.ลาดกระบัง (8ชม.) → monitor

  • โรงผลิตน้ำบางเขน → Dis1/Dis2 → monitor

  • โรงงานผลิตน้ำธนบุรี → โรงพยาบาลศิริราช (2ชม.)

  • โรงผลิตน้ำสามเสน 1-4 → monitor



กลุ่มแม่กลอง (MH):

  • โรงผลิตน้ำมหาสวัสดิ์ → MTR (อุโมงค์) → MDIS (สถานีสูบจ่ายมหาสวัสดิ์) → monitor

  • MDIS → สจ.ราษฎร์บูรณะ (7ชม.), สจ.เพชรเกษม (2ชม.), สจ.ท่าพระ (7ชม.) → monitor




⚠ ข้อจำกัด



• FRC Model อาศัย K ที่ fit จากข้อมูลย้อนหลัง — ถ้าสภาพท่อเปลี่ยนต้อง re-fit
